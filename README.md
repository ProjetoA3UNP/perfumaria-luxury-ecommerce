# E-Commerce de Perfumaria de Luxo e Nicho

## 📌 Sobre o Projeto
[cite_start]Este software é uma plataforma de e-commerce web responsiva especializada em perfumaria de luxo e nicho[cite: 9]. [cite_start]O projeto visa mitigar as barreiras de confiança e sensoriais na compra de fragrâncias online através de curadoria técnica e uma interface intuitiva[cite: 5, 7].

[cite_start]O diferencial da solução está na categorização baseada em famílias olfativas e pirâmides de notas (topo, coração e base), guiando o usuário em uma jornada de descoberta personalizada[cite: 10, 20].

## 🚀 Funcionalidades (Versão 1.0 - MVP)
[cite_start]O escopo inicial foca nas funcionalidades vitais para a operação do negócio[cite: 17]:

* [cite_start]**Módulo de Catálogo:** Exibição de produtos com galeria de fotos, fichas técnicas detalhadas e filtros inteligentes por marca, gênero e família olfativa[cite: 18, 21].
* [cite_start]**Módulo de Vendas:** Carrinho de compras dinâmico, integração com gateways de pagamento (Cartão de Crédito e PIX) e cálculo de frete em tempo real[cite: 22, 24, 25].
* [cite_start]**Módulo de Cliente:** Cadastro, login de usuários e painel "Meus Pedidos" para rastreio[cite: 26, 28].
* [cite_start]**Módulo Administrativo:** Gestão de inventário (CRUD de produtos) e alteração de status de pedidos[cite: 29, 31].

## 🛠️ Especificações Técnicas
* [cite_start]**Arquitetura:** O código segue o padrão **MVC (Model-View-Controller)** para garantir a manutenibilidade[cite: 147].
* [cite_start]**Segurança:** Implementação de criptografia de dados sensíveis e conformidade total com as normas da **LGPD**[cite: 147].
* [cite_start]**Desempenho:** Tempo de carregamento das páginas inferior a 2 segundos[cite: 147].
* [cite_start]**Responsividade:** Interface adaptável para Desktop, Tablet e Mobile[cite: 147].
* [cite_start]**Interoperabilidade:** Integração via API com serviços de logística e pagamento[cite: 147].
