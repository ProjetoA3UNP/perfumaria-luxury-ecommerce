# E-Commerce de Perfumaria de Luxo e Nicho

## 📌 Sobre o Projeto
Este software é uma plataforma de e-commerce web responsiva especializada em perfumaria de luxo e nicho. O projeto visa mitigar as barreiras de confiança e sensoriais na compra de fragrâncias online através de curadoria técnica e uma interface intuitiva.

O diferencial da solução está na categorização baseada em famílias olfativas e pirâmides de notas (topo, coração e base), guiando o usuário em uma jornada de descoberta personalizada.

## 🚀 Funcionalidades (Versão 1.0 - MVP)
O escopo inicial foca nas funcionalidades vitais para a operação do negócio:

* **Módulo de Catálogo:** Exibição de produtos com galeria de fotos, fichas técnicas detalhadas e filtros inteligentes por marca, gênero e família olfativa.
* **Módulo de Vendas:** Carrinho de compras dinâmico, integração com gateways de pagamento (Cartão de Crédito e PIX) e cálculo de frete em tempo real.
* **Módulo de Cliente:** Cadastro, login de usuários e painel "Meus Pedidos" para rastreio.
* **Módulo Administrativo:** Gestão de inventário (CRUD de produtos) e alteração de status de pedidos.

## 🛠️ Especificações Técnicas
* **Arquitetura:** O código segue o padrão **MVC (Model-View-Controller)** para garantir a manutenibilidade.
* **Segurança:** Implementação de criptografia de dados sensíveis e conformidade total com as normas da **LGPD**.
* **Desempenho:** Tempo de carregamento das páginas inferior a 2 segundos.
* **Responsividade:** Interface adaptável para Desktop, Tablet e Mobile.
* **Interoperabilidade:** Integração via API com serviços de logística e pagamento.

## 🚫 Fora de Escopo
Funcionalidades que **não** serão desenvolvidas nesta primeira etapa:
* App Nativo (Acesso exclusivo via navegador).
* Sistema de Assinaturas ("Clube do Perfume").
* Quiz de IA para sugestões personalizadas.
* Vendas B2B (Foco exclusivo no consumidor final).
* Interface multilinguagem (Disponível apenas em PT-BR).
